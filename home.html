Kickstarter Project Success Prediction

About

Kickstarter is a crowdfunding platform that allows creators to raise funds for their projects from a large number of people, often referred to as backers.The crowdfunding industry has gained significant popularity in recent years, and platforms such as Kickstarter have become a hub for creative projects.
All or nothing model: It is a crowdfunding funding approach followed by Kickstarter that only releases funds to a project creator if they meet or exceed their funding goal by the campaign's end. This approach assures backers that their contributions will be used efficiently and that they will get the promised reward.Creators are assured that they will have adequate funds to complete the project.However, it also means that creators won't receive any funds if the goal is not met, and backers won't receive any rewards.

What does this model do?
It enable backers to invest in live projects which are more likely to succeed I have developed a Machine Learning model. This model will enable backers to make data driven investment decisions.

Why Invest in Live Crowdfunding Projects?

  • Low success rate and high risk of failure in crowdfunding projects
  • Disheartening for backers and creators when a project fails to get funded
  • Overfunding of successful projects can divert resources from other deserving projects
  • Wastage of excessive resources and delayed rewards for backers when a project is overfunded
  • Investing in live projects that are more likely to succeed can improve the crowdfunding ecosystem
  • A machine learning model can help predict project success and reduce investment risk for backers

Using the Kickstarter Project Success Prediction Model

  1. Go to the kickstarter website:
    https://www.kickstarter.com/

  2. Searching for projects: Click on the search option


  3. Filter the state of the project: In the more filters dialog box select the live projects option:


  4. Project Selection: Select a project which you feel interested in and which has less than 100% funding. You can try out different combinations of the other filters for selecting a project.
    To boost your project selection by finding a good combination of filters you can check out the following link:
    Analysis and Visualization of Kickstarter data


  5. Give input to the ML model: The ML model is in the predict tab. Once the project is selected the project details should be given as input for the ML model.


    These are the project details which are input for the ML model:
    1: Goal: Funding goal for the project in usd
    2: Projects we love: Yes if it is mentioned along with the Kickstarter icon below the video. If not then the value here is no
    3: Number of Backers: Number of people who have invested in the project
    4: Pledged amount: Funding received by the project in usd
    5: Category: category under which the project falls

    *The input for goal, pledged amount and backers count should be numeric not include any special characters or alphabets like ('$'/','/'.',etc)

  6. Enter the input in the ML model and based on the predicted output make an investment decision accordingly. The ML model is on the Predict page.
